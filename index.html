What is HTML?

HTML, also known as Hypertext Markup Language, is a programming language that is used to describe the structure of web pages. HTML has many useful features such as creating static pages with text, creating headings, creating tables, lists, etcetera, and displaying images.


What is CSS?

CSS, also known as cascading style sheets, is a style sheet language that allows an individual to design a webpage. CSS can do things such as set background colors, font colors and sizes, layout of images, paragraphs, headings, etcetera, and much more.

Why learn CSS and HTML?

Learning CSS and HTML are both both incredibly important programming langauges used primarily for front end web development, which is what the user sees. Learning these langauges will allow you to obtain basic knowledge about creating websites. It is important to understand these languages because they are also needed to understand backend development, which is what the programmer sees.
